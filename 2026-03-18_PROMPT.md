## Circumstance

1. Currently, I've created github repo entitled [autoresearch-skill](https://github.com/wjgoarxiv/autoresearch-skill),
   which is an LLM skill that turns natural-language research goals into autonomous experiment-evaluate-iterate loops -- inspired by Karpathy's autoresearch. Write a research.md, and the agent handles hypothesis generation, experimentation, evaluation, and iteration. Works with Claude Code, Codex CLI, and Gemini CLI.

2. The autoresearch-skill is powerful itself, but I'd like to extend this skill into the next level.
   I've thought that it would be great if there is a skill which can enable to **spawn several autoresearcher (agent, subagent which can perform autoresearch itself)**, discussing each other, making lesson & learn, competing each other, making superior results remaining, inferior results abandoned, etc.
   It's like an some sort of enthusiastic **agora** or **conference**.

It would be great if we can make this kind of format into **SKILL** (make with /skill-creator skill), helping user to spawn **autoresearcher conference**.
This will help user's works into the next level.

3. I'm not sure what would be the ultimate output of this idea. But it would be great if we can concretize this idea into some kind of product to be used.
